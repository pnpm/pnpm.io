---
id: dedupe
title: "pnpm dedupe"
---

Lleva a cabo una instalación eliminando las dependencias anteriores en el archivo de bloqueo si se puede usar una versión más nueva.

## Opciones

### `--check`

Compruebe si ejecutar dedupe resultaría en cambios sin instalar paquetes ni editar el archivo de bloqueo. Sale con un código de estado distinto de cero si es posible realizar cambios.
