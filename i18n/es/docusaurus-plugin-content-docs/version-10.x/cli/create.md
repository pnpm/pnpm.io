id: create
title: "pnpm create"
---

Create a project from a `create-*` or `@foo/create-*` starter kit.

## Ejemplos

```
pnpm create react-app my-app
```

## Opciones

### --allow-build

Added in: v10.2.0

A list of package names that are allowed to run postinstall scripts during installation.
