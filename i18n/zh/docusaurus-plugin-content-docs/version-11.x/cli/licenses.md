## 命令

### list

别名：`ls`

列出已安装包的许可证。

## 配置项

### --dev, -D

仅检查 "devDependencies"。

### --json

以 JSON 格式显示信息。

### --long

显示更多细节（例如仓库链接）。 要显示细节，请传递此选项。

### --no-optional

不检查 optionalDependencies 中的软件包。

### --prod, -P

仅检查 `dependencies` 和 `optionalDependencies`。

### --filter &lt;package_selector\>

[阅读更多有关过滤的内容。](../filtering.md)
