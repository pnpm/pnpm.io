pnpm

Fast, disk space efficient package manager

GitHub docs

features

Speed

pnpm is faster than both npm and Yarn. See benchmarks here.

Efficiency

Packages are saved once per machine. Pnpm never copies, always links. It saves dozens of gigabytes of disk space!

Sanity

pnpm doesn't focus on the reimplementation of other package manager's flaws, like flat dependency trees. But it does focus on complete Node.js compatibility.

Simplicity

pnpm's algorithms are a lot easier than other Node.js package manager's. pnpm doesn't try to replicate the complex algorithms used by npm and yarn for deduplication. pnpm's node_modules design has a sane structure without the problems that npm@2 had.
