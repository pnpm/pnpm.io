# pnpm.io

[![](https://developer.stackblitz.com/img/open_in_codeflow.svg)](https://stackblitz.com/~/github.com/pnpm/pnpm.github.io)

## Testing locally

```
pnpm install
pnpm start
```

## How to publish

Push to the default branch, the website will be deployed automatically.
