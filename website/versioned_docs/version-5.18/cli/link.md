Aliases: `ln`

Makes the current local package accessible system-wide, or in another location.

```text
pnpm link
pnpm link <pkg>
pnpm link <dir>
```

## Options

### --dir \<dir>, -C

* **Default**: Current working directory
* **Type**: Path string

Changes the link location to `<dir>`.
