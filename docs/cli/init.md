title: "pnpm init"
---

Create a `package.json` file.

## Options

### --init-type &lt;type\>

* Default: **commonjs**
* Type: **commonjs**, **module**

Set the module system for the package.

### --init-package-manager

Pin the project to the current pnpm version by adding a "packageManager" field to package.json.
