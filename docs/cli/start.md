---
id: start
title: pnpm start
---

Aliases: `run start`

Runs an arbitrary command specified in the package's `"start"` property of its `"scripts"` object.
If no `"start"` property is specified on the `"scripts"` object, it will run `node server.js`.

## Synopsis

```text
pnpm start [-- &lt;args>...]
```
